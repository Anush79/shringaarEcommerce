## E-Commerce Backend

Ecommerce Application based on React Js. We are selling jewellary items of different categories.

### FEATURES
- Home Page
- Product Listing Page
- Single Product Page
- Filters 
- Wishlist Management
- Cart Management
- Search
- Loading & Alerts
- Authentication
- Sign-up
- Login
- Logout
- Address Management
- Checkout
